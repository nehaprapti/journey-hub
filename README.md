# Tech Zeal

This repository contains the frontend for the Tech Zeal hackathon site (Vite + React + TypeScript).

Quick start

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

Project stack

- Vite
- React (TypeScript)
- Tailwind CSS

Deployment

Build the production bundle with:

```bash
npm run build
```

Then serve the `dist` folder with your preferred static host.
